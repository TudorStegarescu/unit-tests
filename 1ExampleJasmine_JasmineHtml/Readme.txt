Example is based on next tools:
1. UT Framework - Jasmine
2. Test runner - JasmineHtml

Example structure:
1. "js/object.js"  - javascript/jquery functionality
2. "js/tests/object.spec.js" - unit tests for "js/object.js" file

Dependencies are located in helper folder
To start tests - run "JasmineHtml.html"
